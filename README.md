# Secret Notes Calculator App
A calculator which turns into a notes app when a secret code is typed in it.

Has an easter egg in it(hint the square of the number is the addition of the same number)

# To use it:

•Click on **"Code"** download the zip file 

•Extract the zip file

•Open VS code or any text editor you have select the folder secret_notes_app

•click on the **integration.py file** 

•Run it 

Voila there is your calc
